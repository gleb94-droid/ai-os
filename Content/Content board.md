# 🎬 Content board (glebalife)

> **The live content pipeline.** Idea → filmed → posted → numbers (48h) → verdict. Agents keep it current; Gleb drops numbers in chat and they land here. Rhythm rule: **3 things a week, every week** (2 night-shift episodes + 1 value post) — not 20 in a burst.

## 📊 The loop
1. Pick next idea below → film while doing the real work (no extra shoots)
2. Post (hook first line · question/word for comments · price in pinned comment if product)
3. **48h later: drop numbers to the agent** (views · sends · comments · new followers) → logged here
4. Verdict per post: what worked (hook? topic? format?) → next episodes adapt
