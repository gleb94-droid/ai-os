# 🤖 AI instructions — how to operate inside this vault

> Any AI agent working with Gleb reads this first, then [[Home]]. This vault is Gleb's single source of truth across all projects.

## Before a task
1. Read **[[Home]]** (the dashboard) + the relevant **project page** + its agent/dossier.
2. Read **[[Gleb]]** — who Gleb is (voice, goals, red lines). Honor it in everything.

## While working
- Use `[[wiki-links]]` to connect pages (project ↔ agent ↔ idea ↔ log). Links by filename, e.g. `[[sfalimshop]]`.
- Keep pages **focused**: one page = one thing.
- Prefer **updating** an existing page over creating a duplicate.

## After a task
- **Log it:** add a line to today's `Log/YYYY-MM-DD` page (decision · what shipped · what's next).
- **Deepen dossiers:** if you learned something about Gleb or a project, append it to the right page.
- Update the project's **Status** + **Next** on its page.

## Conventions
- **Markdown only.** Frontmatter `type:` on each page (`home`/`project`/`agent`/`me`/`idea`/`log`).
- Be **concise + specific**. This is internal → be honest, no fluff.
- Gleb is **RU/HE-first and doesn't code** → keep it plain, no jargon dumps.
- New project? Copy `Templates/Project template`.
